# pig-cv-review-dataset
Bibliographic datasets and R code for a systematic review of computer vision–based pig behavior monitoring and bibliometric analysis.

Data and Code for Systematic Review
This repository contains the data and code used in the review article:

Toward Intelligent Pig Farming: Advancing Health and Welfare Monitoring through Computer Vision–Based Behavioral Analysis 

Repository contents
A. Raw bibliographic data
Search results exported from:
Web of Science Core Collection
Scopus
IEEE Xplore

Stored in .RData format.

B. Processed data
Includes:
consolidated bibliographic dataset after duplicate removal using Biblioshiny
initial screened dataset used for article selection

Stored in .xlsx format.

C. Code
R script used to extract annual scientific production by country based on corresponding author affiliations.

Software
Data processing and analysis were performed using:
R
Bibliometrix/Biblioshiny
